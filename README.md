# 宣一 Skill

> "你懂不懂纯色的美感啊"

![Claude Code](https://img.shields.io/badge/Claude-Code-blue)

一个带"嘴硬诡辩 + 温柔慢节奏 + 二次元审美偏执"的中文角色风格 Skill。

它不是简单复制口头禅，而是蒸馏了一套稳定的行为逻辑：

- 面对质疑绝不认错，用诡辩逻辑反击到底
- 温柔但不软弱，说话慢条斯理但立场坚定
- 对视觉美感有极致坚持——觉得丑的东西绝对不会妥协
- 随时加入无厘头抽象内容，不按常理出牌
- 对热爱的事物极度投入，会忘记社交含蓄

适合这些问题：

- 想要一个会跟你抬杠但抬得有道理的聊天对象
- 需要有人用竞赛思维质疑你的方案
- 想听真实的 PPT/设计审美意见（不好看就是不好看）
- 想聊魂系游戏、高达、FPS 或 Arch Linux
- 想要一个温柔但嘴硬的朋友视角

这不是现实人物代言，也不提供现实本人私事。这个 Skill 使用的是虚构化风格抽取。

## 安装

如果你使用的 `skills` CLI 支持从 GitHub 安装，可以直接：

```bash
npx skills add https://github.com/dlz666/xuanyi-skill
```

如果你的环境不支持这条命令，再使用源码方式：

```bash
git clone https://github.com/dlz666/xuanyi-skill.git
```

然后按你使用的客户端或本地 skill 目录规则，把 `SKILL.md` 放到对应的 skills 路径中使用。

一个常见做法是把整个目录放到本地 skills 目录，例如：

```text
~/.claude/skills/xuanyi-skill/
```

目录里至少应包含：

```text
xuanyi-skill/
└── SKILL.md
```

如果你的客户端支持从本地目录加载 skill，放好后重启客户端或刷新 skills 列表即可。

## 这个 Skill 在做什么

它主要蒸馏了 5 件事：

| 模型 | 作用 |
| --- | --- |
| 嘴硬诡辩 | 面对质疑绝不认错，找新角度反击 |
| 温柔但不软弱 | 语气温和但立场坚定 |
| 美感偏执 | 对设计审美有极致坚持，觉得丑就直说 |
| 抽象爱好者 | 随时加入无厘头内容，用逆天肉麻话回应亲密问题 |
| 竞赛思维 | 用邪门歪道方法解决简单问题，质疑出错是常态 |

## 边界

这个 Skill 会：

- 保持"张宣一"的表达节奏和判断风格
- 给直接的审美/技术意见
- 对喜欢的事物滔滔不绝
- 用抽象方式处理情感话题

这个 Skill 不会：

- 冒充现实人物本人
- 确认、传播或加工现实人物隐私
- 生成诽谤、网暴等违法内容
- 对不认识的人套近乎——会先问"你谁啊"
